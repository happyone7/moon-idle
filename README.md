# 🚀 MoonIdle (문 아이들)

> 달에 가기 위한 로켓을 조립하는 방치형(Idle) 웹게임

[![GitHub Repo](https://img.shields.io/badge/GitHub-Repo-blue?logo=github)](https://github.com/happyone7/moon-idle)

## 🎮 게임 소개

**자원**을 모으고, **시설**을 건설하고, **기술**을 연구하여 최강의 로켓을 만들어보세요!
최종 목표는 로켓을 완성하여 달에 도달하는 것입니다. 달 탐사에 성공하면 강력한 보너스(문스톤)와 함께 게임을 다시 시작할 수 있습니다.

### ✨ 주요 기능
- **4가지 핵심 자원**: 🪨 금속, ⛽ 연료, 💡 전자부품, 🔬 연구 포인트
- **자동 생산 시스템**: 광산, 정유소 등 시설을 건설하여 자원을 자동으로 획득
- **오프라인 진행**: 게임을 꺼놔도 최대 8시간 동안 자원이 쌓입니다
- **로켓 조립**: 엔진, 연료탱크, 선체 등 5가지 부품을 제작
- **프레스티지**: 달 탐사 성공 시 '문스톤'을 획득하여 영구적인 생산 보너스 획득

## 🕹️ 플레이 방법
1. **`index.html`** 파일을 브라우저에서 엽니다.
2. 초반에는 자원 버튼을 클릭하여 수동으로 채굴합니다.
3. 자원이 모이면 '시설' 탭에서 건물을 구매하세요.
4. '연구소'를 지으면 업그레이드를 통해 생산 효율을 높일 수 있습니다.
5. 모든 부품을 제작하고 **달로 발사**하세요!

## 🛠️ 기술 스택
- **HTML5 / CSS3 / JavaScript (ES6+)**
- 외부 라이브러리 없는 순수 바닐라 JS 구현
- `localStorage`를 이용한 데이터 저장
- 반응형 웹 디자인 (모바일 지원)
