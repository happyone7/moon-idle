## 조사 결과 요약 (수정 근거)

### Issue A: 역슬래시 → 원화(₩) 표시
- **위치**: `js/tabs/launch.js`
- **원인**: JS 문자열에서 `\\` 이스케이프로 삽입된 `\` (U+005C)가 한국어 Windows 폰트에서 `₩`로 렌더링
- **영향 부위**: 발사 애니메이션 로켓 아트, 오버레이 로켓 아트, 폭발 Frame 1

### Issue B: 건물 벽 돌출 (폭 불일치)
- **위치**: `js/world.js`
- **원인**: ASCII 박스 테두리 폭과 기반부(████) 폭이 불일치 — 벽이 기반부보다 넓어 "공중에 떠 있는" 효과 발생
- **영향 건물**: housing, ops_center, supply_depot, research_lab, r_and_d, elec_lab, solar_array(3패널), launch_pad

## 작업 항목

### P6-1 (프로그래밍팀장): launch.js 역슬래시 수정
**파일**: `js/tabs/launch.js`
**내용**: 모든 ASCII 아트 문자열에서 `\\` → `╲` (U+2572, BOX DRAWINGS LIGHT DIAGONAL UPPER LEFT TO LOWER RIGHT) 대체

구체적 수정 위치:
1. `_runLaunchAnimation` animWrap.innerHTML 로켓 아트 (`/|\\`, `/ | \\`, `/  |  \\`)
2. `_showLaunchOverlay` loRocket.textContent (`/|\\`, `/ | \\`, `/  |  \\`)
3. FAIL_FRAMES Frame 1 (`/\\!\\`, `/ !!  \\`, `/ WARN \\`)

**주의**: `world.js`의 launchpad 아트는 이미 `╲` (U+2572) 사용 중 — `launch.js`만 수정
