## 1. v7a 생산 허브 (Production Hub) — 갭 목록

### 1.1 월드 씬 표시 방식 (핵심 차이)
| 항목 | v7a 목업 | 현재 빌드 | 심각도 |
|------|----------|-----------|--------|
| 월드 렌더링 | 메인 콘텐츠 영역 — 2800px 스크롤 가능한 세계 | `#world-bg` fixed 배경, opacity 0.15~0.5, blur | ⚠️ HIGH |
| 월드 위치 | 3-column 레이아웃의 center 영역 (flex:1) | z-index:0 고정 배경, 탭 콘텐츠 뒤에 깔림 | ⚠️ HIGH |
| 스크롤 | `#ws` 수평 스크롤 (cursor:grab/grabbing 드래그) | 배경이라 스크롤 불가 | ⚠️ HIGH |

**평가**: 월드 씬은 v7a에서 게임의 핵심 비주얼 요소. 현재 빌드에서는 흐릿한 배경으로 격하됨. **가장 큰 차이점.**

## 8. 작업 지시 방향 (기획팀장 + TA)

### 기획팀장 담당 (레이아웃 + 기능 명세)
1. **조립 탭 3-column 레이아웃** 명세 작성 (v7c 기반)
2. **로켓 클래스 선택기** 기획 (NANO~SUPER HEAVY, 해금 조건)
3. **BOM 테이블 구조** 명세 (부품 계층, 소재별 상태)
4. **7단계 조립 진행** 명세 (각 스테이지 명칭/조건)
5. **임무 탭 서브탭** 명세 (4개 서브탭 콘텐츠 구조)
6. **업적 시스템** 명세 (업적 목록, 조건, 보상)
7. **프레스티지 스타 트리** 명세 (업그레이드 경로, 비용, 효과)

### TA 담당 (비주얼 + CSS)
1. **프로그레스 바 shimmer + leading edge** CSS 적용 (전 탭)
2. **연구 카드별 ASCII 아트** 제작 (기술별 전용 ASCII)
3. **조립 진행 ASCII 표시** 제작 (STAGE 01~07)
4. **조립 도구 패널** ASCII 아트 제작
5. **임무 페이즈별 ASCII 씬** 5개 제작 (프로토타입→달 표면)
6. **리스크 태그 펄스** 애니메이션 적용
7. **앰버 프로그레스 바** 변형 CSS
8. **ERA 뱃지 + 커서 블링크** 상단 바 요소
