## 1. 진행 단계 (5 페이즈)

### 개요
MoonIdle의 진행은 크게 5개의 에라(Era)로 나뉜다. 각 에라는 독립된 기술적 도전과 감정적 경험을 제공하며, 플레이어는 이전 에라의 지식을 축적하며 다음 에라로 진입한다.

---

### Phase 1: 프로토타입 에라 (Prototype Era)
**슬로건**: "하늘에 닿는 첫 번째 시도"

#### 진입 조건
- 게임 시작 시 자동 시작
- 초기 자원: 철광석 100단위, 자금 ₩500,000

#### 에라 컨셉
플레이어는 허름한 창고형 공장에서 시작한다. 목표는 단 하나 — 무언가를 하늘로 보내는 것. 실패가 기본값이며, 실패에서 배우는 과정 자체가 게임플레이다. 이 시기의 로켓은 "사운딩 로켓(Sounding Rocket)" 수준으로, 실제 우주 진입보다는 고고도 대기권 탐사에 집중한다.

#### 이 에라에서 등장하는 핵심 자원
| 자원 | 한글명 | 역할 |
|------|--------|------|
| Iron Ore | 철광석 | 기본 구조재 원료 |
| Steel | 강철 | 1차 가공 금속 |
| RP-1 Kerosene | RP-1 케로신 | 초기 연료 |
| LOX (Liquid Oxygen) | 액체 산소 | 산화제 |
| Research Points | 연구 포인트 | 기술 해금 화폐 |

#### 핵심 시설 (이 에라에서 해금)
- 철광석 채굴기 (Iron Ore Mine) Lv.1
- 용광로 Lv.1 (강철 생산)
- RP-1 정제 시설 Lv.1
- 기초 연구소 Lv.1
- 로켓 조립동 Lv.1 (소형)
- 발사대 Lv.1 (소형 콘크리트)

#### 핵심 마일스톤
| 마일스톤 | 목표 고도 | 비고 |
|----------|-----------|------|
| First Ignition | 지상 이탈 | 성공 여부 불문, 점화만 해도 달성 |
| 1km Club | 1km | 첫 번째 진짜 로켓 |
| Boundary Pusher | 10km | 날씨 영향권 이탈 |
| Stratosphere | 20km | 오존층 진입 |
| Felix Line | 39km | 성층권 최상부 (Felix Baumgartner 기준) |

#### 플레이어 감정 경험
- **초반**: 불안과 흥분. "내 로켓이 터질 것 같아" — 실제로 자주 터진다
- **중반**: 점진적 자신감. 실패 원인을 파악하고 개선하는 쾌감
- **후반**: 첫 10km 달성의 감동. "내가 진짜로 뭔가 만들었다"

#### 에라 종료 조건
- 로켓이 10km 고도 달성
- RP-1/LOX 추진 엔진 기술 해금
- 기초 연구소에서 "대기권 돌파 연구" 완료

### Phase 2: 준궤도 에라 (Suborbital Era)
**슬로건**: "카르만 선을 넘어라"

#### 진입 조건
- Phase 1 완료 (10km 달성)
- 알루미늄 제련소 건설
- "구조 경량화 I" 기술 보유

#### 에라 컨셉
카르만 선(Kármán Line, 100km)을 넘어 기술적으로 "우주"에 도달하는 것이 목표. 이 시기부터 진공 환경에서의 엔진 작동, 열 관리, 재진입 문제가 등장한다. "준궤도(Suborbital)"는 궤도 속도에는 미치지 못하지만 우주 고도에 도달하는 비행 궤적이다. V-2 로켓, Mercury-Redstone, SpaceShipOne이 이 범주에 속한다.

#### 이 에라에서 등장하는 핵심 자원
| 자원 | 한글명 | 역할 |
|------|--------|------|
| Bauxite Ore | 보크사이트 광석 | 알루미늄 원료 |
| Aluminum Alloy (Al-7075) | 알루미늄 합금 | 경량 구조재 |
| CFRP | 탄소섬유 강화 플라스틱 | 고강도 경량 복합재 |
| Launch Data | 발사 데이터 | 실패에서 얻는 경험치 |
| Reputation | 명성 | 투자자 유치, 자금 획득 |

#### 핵심 시설 (이 에라에서 해금)
- 보크사이트 채굴기 Lv.1
- 알루미늄 제련소 Lv.1
- CFRP 적층 공장 Lv.1
- 추진 연구소 Lv.1
- 로켓 조립동 Lv.2 (중형)
- 발사대 Lv.2 (강화 콘크리트, 냉각수 시스템)
- 추적 안테나 Lv.1

#### 핵심 마일스톤
| 마일스톤 | 목표 | 비고 |
|----------|------|------|
| Mesosphere Entry | 50km | 중간권 진입 |
| Kármán Line | 100km | "공식 우주" 달성 |
| Tourist Altitude | 120km | Blue Origin 기준선 |
| Weather Eye | 200km | 대기 없는 진공 확인 |
| Suborbital Return | 100km + 회수 성공 | 재사용 첫 단계 |

#### 플레이어 감정 경험
- **카르만 선 돌파 순간**: 화면에 "YOU REACHED SPACE" 연출 — 게임 최초의 감정적 클라이맥스
- **재진입 실패**: "다 됐는데..." — 재진입 열 관리의 어려움을 실감
- **첫 회수 성공**: 재사용의 경제성을 이해하기 시작

#### 에라 종료 조건
- 카르만 선 100km 달성 2회 이상
- 알루미늄/CFRP 생산 라인 안정화
- 추진 연구소에서 "터보펌프 사이클" 연구 완료

### Phase 3: 궤도 에라 (Orbital Era)
**슬로건**: "7.8km/s의 의미"

#### 진입 조건
- Phase 2 완료
- "터보펌프 사이클" 기술 보유
- 추진제 저장 탱크 팜 건설
- 자금 ₩50,000,000 이상

#### 에라 컨셉
진짜 우주 비즈니스의 시작. 저궤도(LEO, 400km 이상)에 진입하려면 약 7.8 km/s의 속도가 필요하다. 이것은 단순히 "높이"의 문제가 아니라 "속도"의 문제다. 이 에라에서 플레이어는 다단 로켓(Multi-stage rocket), 페이로드 분리, 위성 배치를 배운다. 성공적인 위성 발사는 **반복 수입원**이 된다.

#### 이 에라에서 등장하는 핵심 자원
| 자원 | 한글명 | 역할 |
|------|--------|------|
| Titanium Ore | 티타늄 광석 | 고온 구조재 원료 |
| Titanium Alloy (Ti-6Al-4V) | 티타늄 합금 | 엔진 터빈, 고온부 |
| Inconel 718 | 인코넬 718 | 연소실, 노즐 소재 |
| Copper Wire | 구리 배선 | 전자 시스템 |
| Circuit Board | 회로 기판 | 항전 장비 |
| LH2 (Liquid Hydrogen) | 액체 수소 | 고성능 연료 (Isp 450+) |

#### 핵심 시설 (이 에라에서 해금)
- 티타늄 광산 Lv.1
- 티타늄 정제로 Lv.1
- 정밀 기계 가공소 Lv.1
- 전자 부품 제작소 Lv.1
- LH2 생산소 Lv.1
- 항법 연구소 Lv.1
- 로켓 조립동 Lv.3 (대형 VAB)
- 발사대 Lv.3 (LH2 지원, 음향 억제)
- 기상 관측소 Lv.1

#### 핵심 마일스톤
| 마일스톤 | 목표 | 비고 |
|----------|------|------|
| First Stage Separation | 1단 분리 성공 | 다단 로켓 첫 달성 |
| Orbital Velocity | 7.8 km/s 달성 | "진짜 궤도" |
| LEO Stable Orbit | 400km 90분 주기 궤도 | ISS 고도 |
| First Satellite | 위성 1기 배치 | 반복 수입 시작 |
| Constellation Start | 위성 3기 동시 운용 | 수입 ×3배 |
| Reusability I | 1단 부스터 회수 | 비용 -40% |

#### 플레이어 감정 경험
- **첫 궤도 진입**: "드디어 지구를 도는 뭔가를 만들었다" — 엔드게임이 보이기 시작
- **위성 수입**: 처음으로 게임이 "돈을 버는" 느낌
- **부스터 회수**: SpaceX 팰컨 9 착륙 장면을 내 손으로 만드는 감동

#### 에라 종료 조건
- LEO 안정 궤도 (400km 이상, 주기 90분) 달성
- 위성 1기 이상 배치 운용 중
- "달 궤적 계산" 기술 연구 완료

### Phase 4: 시스루나 에라 (Cislunar Era)
**슬로건**: "지구 중력권을 탈출하다"

#### 진입 조건
- Phase 3 완료
- 헤비 로켓 설계 해금
- "달 궤적 계산" + "심우주 통신" 기술 보유
- 명성(Reputation) 500 이상

#### 에라 컨셉
"Cislunar"(시스루나)는 지구와 달 사이의 공간을 의미한다. 트랜스 루나 인젝션(TLI, Trans-Lunar Injection)을 통해 달로 향하는 궤도에 진입하고, 달 주회 궤도(Lunar Orbit, LO)를 달성하는 것이 이 에라의 목표다. Apollo 시대의 엔지니어링 문제들 — 하우만 전이, LOI(달 궤도 진입 번), 심우주 통신 — 이 실제 게임 메카닉으로 등장한다.

#### 이 에라에서 등장하는 핵심 자원
| 자원 | 한글명 | 역할 |
|------|--------|------|
| Silicon | 실리콘 | 태양전지, 반도체 |
| N2O4 | 사산화이질소 | 심우주 추진제 (산화제) |
| UDMH | 비대칭 디메틸히드라진 | 심우주 추진제 (연료) |
| Hypergolic Thruster Kit | 하이퍼골릭 추력기 키트 | 자세 제어, 궤도 수정 |
| Deep Space Comms Array | 심우주 통신 배열 | 달 거리 통신 유지 |

#### 핵심 시설 (이 에라에서 해금)
- 실리콘 정제 시설 Lv.1
- N2O4/UDMH 하이퍼골릭 추진제 플랜트 Lv.1
- 심우주 추적 안테나 (DSN급) Lv.1
- 소재 연구소 Lv.2 (고온 TPS 연구)
- 로켓 조립동 Lv.4 (슈퍼 헤비 클래스)
- 의료 센터 Lv.1 (유인 미션 준비)

#### 핵심 마일스톤
| 마일스톤 | 목표 | 비고 |
|----------|------|------|
| Earth Escape | 지구 탈출 속도 11.2 km/s | 달 방향 TLI |
| Lunar Flyby | 달 근접 통과 (1,000km 이내) | 중력 보조 첫 경험 |
| LOI Success | 달 궤도 진입 번 성공 | 달 궤도 포획 |
| Lunar Orbit | 달 주위 100km 안정 궤도 | 착륙 준비 |
| Lunar Relay Sat | 달 통신 중계 위성 배치 | 착륙 통신 확보 |

#### 플레이어 감정 경험
- **달이 화면에 처음 크게 보이는 순간**: 시각적 연출 클라이맥스 2
- **LOI 번 긴장감**: "이 연소가 실패하면 달에서 못 돌아온다" (Apollo 13 감정)
- **달 궤도 달성**: 목표가 진짜로 손에 잡힌다

#### 에라 종료 조건
- 달 안정 궤도 달성 (100±20km, 주기 약 2시간)
- 달 통신 중계 위성 배치 완료
- "달 착륙선 설계" 기술 연구 완료

### Phase 5: 달 표면 에라 (Lunar Surface Era)
**슬로건**: "인류의 두 번째 고향"

#### 진입 조건
- Phase 4 완료
- "달 착륙선 설계" + "ISRU 기초" 기술 보유
- 슈퍼 헤비 로켓 클래스 해금

#### 에라 컨셉
달 착륙은 게임의 최종 목표이자 새로운 시작이다. ISRU(In-Situ Resource Utilization, 현지 자원 활용)를 통해 달 표면의 레골리스(Regolith)에서 산소와 금속을 추출하고, 달 기지 건설의 첫 단계를 구축한다. 이 에라에서 게임은 "달 식민지"라는 더 큰 그림으로 전환된다.

#### 이 에라에서 등장하는 핵심 자원
| 자원 | 한글명 | 역할 |
|------|--------|------|
| Lunar Regolith | 달 레골리스 | ISRU 원료, 달 표면 재료 |
| Lunar Oxygen (LOX-ISRU) | 달 산소 | 현지 추진제 생산 |
| Lunar Aluminum | 달 알루미늄 | 현지 구조재 |
| Helium-3 | 헬륨-3 | 미래 연료 (게임 내 최고가 자원) |
| Water Ice | 물 얼음 | 달 극지 자원, 음료·전기분해 |

#### 핵심 시설 (이 에라에서 해금, 달 표면에 건설)
- 달 착륙 패드 (Lunar Landing Pad) Lv.1
- ISRU 플랜트 — 레골리스 처리 Lv.1
- 달 거주 모듈 Lv.1 (기지 허브)
- 달 로버 차고 Lv.1
- 달 표면 태양광 어레이 Lv.1
- 달 드릴 (극지 얼음 채굴) Lv.1

#### 핵심 마일스톤
| 마일스톤 | 목표 | 비고 |
|----------|------|------|
| Soft Landing | 달 표면 무사 착륙 (무인) | 게임 핵심 목표 |
| Sample Return | 달 토양 샘플 귀환 | 과학 데이터 최대치 |
| ISRU First Output | 달 산소 1단위 생산 | 현지 자원 활용 시작 |
| Lunar Base Alpha | 거주 모듈 + 에너지 + ISRU 완비 | "기지" 공식 인정 |
| Crewed Landing | 유인 착륙 성공 | 게임 최종 엔딩 컷신 |
| He-3 Discovery | 헬륨-3 발견 | 프레스티지 보너스 잠금 해제 |

#### 플레이어 감정 경험
- **착륙 성공 순간**: Apollo 11 실황 방송 오마주 연출 ("Eagle has landed")
- **ISRU 첫 산소 생산**: "이제 달이 내 것이다" — 자급자족의 감동
- **유인 착륙**: 게임 엔딩. 단, 프레스티지 후 "화성 미션" 티저 화면

### Tier 1: 원자재 (Raw Materials)

## 5. 로켓 구성 시스템

### 설계 철학
플레이어는 로켓을 부품 단위로 설계하지 않는다. 대신 **로켓 클래스**와 **설정 옵션**을 선택하는 방식으로 구성한다. 이 방식은 기술 현실성을 유지하면서도 idle 게임의 접근성을 보장한다.

---

### 로켓 클래스 (Rocket Classes)

#### Nano Class: 나노 로켓
- **대표 실물 참조**: Rocket Lab Electron 소형, V-2 (역사적)
- **해금 조건**: Phase 1 시작
- **스테이지 구성**: 1단 (고체 또는 LOX/RP-1)
- **최대 고도**: 100km (준궤도)
- **페이로드 용량**: 50kg 이하
- **조립 시간**: VAB Lv.1에서 30분
- **재료 비용**: 강철 80 + RP-1 200 + LOX 400
- **용도**: Phase 1-2 발사, 기초 발사 데이터 수집

#### Small Class: 소형 로켓
- **대표 실물 참조**: Rocket Lab Electron, Firefly Alpha
- **해금 조건**: Phase 2 + "알루미늄 구조 I" 기술
- **스테이지 구성**: 1~2단 (LOX/RP-1)
- **최대 고도**: 500km (LEO)
- **페이로드 용량**: 300kg (LEO 기준)
- **조립 시간**: VAB Lv.2에서 40분
- **재료 비용**: Al합금 150 + CFRP 30 + RP-1 500 + LOX 1,000

#### Medium Class: 중형 로켓
- **대표 실물 참조**: SpaceX Falcon 9, ULA Atlas V
- **해금 조건**: Phase 3 + "터보펌프 사이클" 기술
- **스테이지 구성**: 2단 (LOX/RP-1, 상단 LOX/LH2 선택 가능)
- **최대 고도**: 1,200km (LEO~MEO)
- **페이로드 용량**: 5,000kg (LEO 기준)
- **조립 시간**: VAB Lv.3에서 50분
- **재료 비용**: Al합금 400 + CFRP 100 + Ti합금 30 + 엔진 조립체 9기

#### Heavy Class: 헤비 로켓
- **대표 실물 참조**: SpaceX Falcon Heavy, NASA SLS Block 1
- **해금 조건**: Phase 4 + "단계 연소 사이클" 기술
- **스테이지 구성**: 2단 + 측면 부스터 2기
- **최대 고도**: 달 궤도 (384,400km 도달 가능)
- **페이로드 용량**: 26,000kg (LEO) / 3,500kg (TLI)
- **조립 시간**: VAB Lv.4에서 90분
- **재료 비용**: Al합금 1,200 + CFRP 400 + Ti합금 150 + 인코넬 80 + 엔진 27기

#### Super Heavy Class: 슈퍼 헤비 로켓
- **대표 실물 참조**: SpaceX Starship/Super Heavy, NASA Saturn V (역사적)
- **해금 조건**: Phase 4-5 + "풀 흐름 단계 연소" + "달 착륙선 설계" 기술
- **스테이지 구성**: 2단 (1단: 부스터 33기, 2단: 우주선 + 착륙선)
- **최대 고도**: 달 착륙
- **페이로드 용량**: 150,000kg (LEO) / 달 착륙 가능 탑재
- **조립 시간**: VAB Lv.5에서 180분
- **재료 비용**: 엄청난 양 (Phase 5 수준 생산 라인 필요)

## 6. 발사 및 실패 시스템

### 철학: "실패는 데이터다"
초기 게임에서 발사 성공률은 30~50%에 불과하다. 모든 실패는 "발사 데이터(Launch Data)"를 제공하며, 이 데이터가 누적될수록 성공률이 높아진다. 이는 실제 로켓 개발 역사 — V-2에서 팰컨 1까지 — 를 게임 메카닉으로 구현한 것이다.

---

### 발사 단계 (Launch Phases)
발사는 4개의 단계로 진행되며, 각 단계마다 고유한 실패 가능성이 있다.

```
[T-10분: 발사 전 점검] → [T-0: 점화] → [Max-Q: 최대 동압] → [1단 분리]
→ [2단 점화 (해당 시)] → [궤도 투입 (해당 시)] → [페이로드 분리]
→ [부스터 회수 (해당 시)]
```

---

### 실패 모드 8종 (Failure Modes)

#### 1. Max-Q 구조 실패 (Structural Failure at Max-Q)
- **발생 시점**: 이륙 후 약 1~2분 (최대 동압 구간, 고도 10~15km)
- **원인**: 동체 강도 부족, CFRP 품질 미달
- **게임 내 트리거**: 동체 품질 등급 < 로켓 클래스 요구치
- **실패 확률**: 기본 15% → 구조 기술 업그레이드로 감소
- **발사 데이터 보상**: 8점 (높음 — 위험 구간 통과 정보)
- **부분 성공**: 없음 (즉각 폭발)

#### 2. 엔진 실패 (Engine Failure)
- **발생 시점**: 이륙 직후 (T+0~T+30초) 또는 2단 점화 시
- **원인**: 터보펌프 이상, 연소 불안정, 인젝터 막힘
- **게임 내 트리거**: 엔진 조립체 품질 < 70%
- **실패 확률**: 기본 20% → 추진 기술 업그레이드로 감소
- **발사 데이터 보상**: 10점 (매우 높음)
- **부분 성공**: 나머지 엔진으로 일부 고도 달성 시 부분 보상

#### 3. 유도 실패 (Guidance Failure)
- **발생 시점**: 비행 중 언제든 (확률 분포: 전 구간 균등)
- **원인**: 자이로스코프 이상, 소프트웨어 버그, 전자기 간섭
- **게임 내 트리거**: 회로 기판 등급 < 항전 요구치
- **실패 확률**: 기본 25% → 항전 기술 업그레이드로 감소
- **발사 데이터 보상**: 7점
- **부분 성공**: 목표 고도 50% 도달 시 부분 보상 50%

#### 4. 연료 고갈 (Fuel Depletion)
- **발생 시점**: 목표 고도 도달 직전 (마지막 10% 구간)
- **원인**: 연료 계산 오류, 탱크 누설, 예상보다 높은 항력
- **게임 내 트리거**: 페이로드 중량 > 설계 한도
- **실패 확률**: 기본 10% → 추진제 탱크 용량 업그레이드로 감소
- **발사 데이터 보상**: 5점
- **부분 성공**: 목표의 85% 이상 도달 시 부분 성공 75%

#### 5. 스테이지 분리 실패 (Staging Failure)
- **발생 시점**: 1단~2단 분리 순간
- **원인**: 분리 볼트 불량, 화약 불발, 기계 잠금
- **게임 내 트리거**: 다단 로켓 처음 2~5회 (학습 곡선)
- **실패 확률**: 기본 18% (2단 로켓 처음 시도) → 경험 축적으로 감소
- **발사 데이터 보상**: 12점 (특히 중요한 기술적 교훈)
- **부분 성공**: 1단만으로 도달 가능한 고도까지 부분 성공

#### 6. 재진입 실패 (Re-entry Failure)
- **발생 시점**: 대기권 재진입 시 (귀환 미션만 해당)
- **원인**: 열차폐 부족, 진입각 오류 (너무 가파름/얕음)
- **게임 내 트리거**: 열 관리 기술 부족 + 재진입 미션
- **실패 확률**: 기본 30% → 열 관리 기술로 감소
- **발사 데이터 보상**: 15점 (매우 중요)
- **부분 성공**: 페이로드 일부 회수 (내열 캡슐 기술 있을 경우)

#### 7. 통신 두절 (Communications Loss)
- **발생 시점**: 추적 안테나 범위 이탈 시
- **원인**: 안테나 범위 부족, 지평선 너머
- **게임 내 트리거**: 목표 고도 > 안테나 Lv. 범위
- **실패 확률**: 범위 초과 비율에 비례 (0~100%)
- **발사 데이터 보상**: 3점 (낮음 — 엔지니어링 실패 아님)
- **부분 성공**: 통신 두절 전 도달 고도까지 부분 보상

#### 8. 범위 안전 비상 중단 (Range Safety Abort)
- **발생 시점**: 이상 비행 궤적 감지 시
- **원인**: 유도 시스템 이상으로 예상 궤도 이탈 → 자동 자폭
- **게임 내 트리거**: 유도 실패의 "안전 종료" 버전
- **실패 확률**: 기본 8% (유도 실패 중 일부)
- **발사 데이터 보상**: 6점
- **부분 성공**: 없음 (안전 중단이므로 의도적 종료)

---

### 실패 확률 계산식

```
총 실패 확률 = Σ(각 실패 모드 기본 확률) × (1 - 기술 보정) × (1 - 데이터 보정) × 날씨 계수

기술 보정 = (관련 기술 레벨 합계) × 0.05 (최대 60%)
데이터 보정 = (보유 Launch Data) × 0.001 (최대 40%)
날씨 계수 = 1.0 (맑음) ~ 2.0 (폭풍 경보)
```

**예시**: Phase 2 초기 소형 로켓 발사
- 기본 실패 모드 합계: Max-Q 15% + 엔진 20% + 유도 25% = 60%
- 기술 보정: 기술 레벨 합계 6 → ×0.70
- 데이터 보정: Launch Data 30점 → ×0.97
- 날씨 계수: 맑음 → ×1.0
- **최종 실패 확률**: 60% × 0.70 × 0.97 ≈ 40.7%

### 8.1 오프라인 생산 (Offline Production)

#### 규칙
- **최대 오프라인 적립 시간**: 8시간
- **오프라인 효율**: 온라인 대비 70% (완전 방치 페널티)
- **자동 관리자 보유 시**: 오프라인 효율 85%로 상승
- **저장 한도**: 오프라인 적립은 시설 저장 한도에 구애받지 않음 (단, 8시간치 상한)

### 8.2 자동화 관리자 시스템 (Auto-Manager System)

플레이어는 가상의 "엔지니어"를 고용하여 시설 클릭 없이 자동 생산을 달성한다.

#### 관리자 종류

| 관리자 | 한글명 | 고용 비용 | 담당 시설 | 효과 |
|--------|--------|-----------|-----------|------|
| 채굴 감독관 | Mine Supervisor | 엔지니어 토큰 3개 | 채굴 카테고리 전체 | 클릭 없이 자동 생산 |
| 제련 엔지니어 | Refinery Engineer | 엔지니어 토큰 3개 | 정제 카테고리 전체 | 클릭 없이 자동 생산 |
| 생산 관리자 | Production Manager | 엔지니어 토큰 5개 | 제조 카테고리 전체 | 자동 + 효율 +10% |
| 연료 기술자 | Propellant Tech | 엔지니어 토큰 4개 | 추진제 카테고리 전체 | 자동 + 저장 효율 +15% |
| 수석 연구원 | Chief Researcher | 엔지니어 토큰 6개 | 연구 카테고리 전체 | 자동 + RP +20% |
| 발사 감독 | Launch Director | 엔지니어 토큰 8개 | 발사 준비 자동화 | 조립→연료→발사 완전 자동 |

#### 관리자 업그레이드
각 관리자는 자금으로 업그레이드 가능:
- Lv.1: 기본 자동화
- Lv.2: 자동화 + 효율 +10%
- Lv.3: 자동화 + 효율 +25% + 오프라인 효율 향상

## 9. UI 및 비주얼 함의

### 9.1 메인 허브 화면 (Main Hub Screen)

#### 표시해야 할 데이터
- 현재 에라 (Phase 1~5) 및 진행률
- 핵심 자원 6종 현재량/저장한도 (Tier별 색상 구분)
- 자금 현재량 + 분당 수입
- 명성 게이지 (0~1000)
- 활성 시설 수 / 총 시설 수
- 다음 마일스톤 미리보기

#### 필요한 인터랙션
- 각 자원 탭 → 해당 생산 시설로 바로 이동
- 자금 클릭 → 수입/지출 내역 팝업
- 에라 아이콘 클릭 → 진행 단계 상세 화면

#### 비주얼 피드백
- 자원 넘침(오버플로): 저장 바가 빨간색으로 깜빡임 + 손실량 표시
- 자금 부족: 건설/업그레이드 버튼 비활성화 + 부족 금액 표시
- 신규 기술 해금: 기술 트리 아이콘에 알림 배지

---

### 9.2 시설 관리 화면 (Facility Management Screen)

#### 표시해야 할 데이터
- 시설 목록 (카테고리 필터 탭: 채굴 / 정제 / 제조 / 추진제 / 연구 / 운용)
- 각 시설: 이름, 레벨, 현재 생산율, 입력/출력 자원
- 업그레이드 비용 + 업그레이드 후 효과 미리보기
- 시설 카드에 "병목" 경고 (입력 자원 부족 시 빨간 테두리)

#### 필요한 인터랙션
- 시설 카드 클릭 → 상세 정보 + 업그레이드 버튼
- 업그레이드 버튼: 자금 충분 시 파란색, 부족 시 회색 + 부족 금액
- "모두 자동화" 토글 버튼 (관리자 보유 시 활성화)

#### 비주얼 피드백
- 생산 중: 시설 아이콘 애니메이션 (회전, 불빛 깜빡임)
- 중단(재료 부족): 아이콘 회색화 + "재료 부족" 툴팁
- 업그레이드 효과: "+" 파티클 이펙트

---

### 9.3 로켓 구성 화면 (Rocket Configuration Screen)

#### 표시해야 할 데이터
- 로켓 클래스 선택 (잠금/해금 상태 시각화)
- 선택한 클래스의 스펙: 페이로드 용량, 최대 고도, 성공 확률 예측
- 페이로드 선택 드롭다운
- 엔진 선택 (잠금/해금)
- 조립에 필요한 재료 체크리스트 (보유량/필요량 비교)
- 예상 발사 성공률 (퍼센트 + 주요 위험 요인 표시)

#### 필요한 인터랙션
- 로켓 클래스 선택 → 옵션 동적 변경
- 재료 부족 항목 클릭 → 해당 생산 시설 화면으로 이동
- "조립 시작" 버튼 → VAB 애니메이션 + 조립 시간 카운트다운

#### 비주얼 피드백
- 로켓 미리보기 이미지: 선택한 클래스에 따라 변경
- 성공률 게이지: 초록(80%+) → 노랑(50~80%) → 빨강(50% 미만)
- 위험 요인 목록: 아이콘 + 짧은 설명 (예: "열 관리 기술 부족 — 재진입 실패 위험")

### 9.4 발사 화면 (Launch Sequence Screen)

#### 표시해야 할 데이터
- 카운트다운 타이머 (T-10:00 → T-0)
- 실시간 고도계 (km)
- 실시간 속도계 (km/s)
- 연료 잔량 게이지
- 발사 단계 진행 바 (점화 → Max-Q → 1단 분리 → 2단 점화 → 목표 도달)
- 통신 신호 강도 게이지

#### 필요한 인터랙션
- "발사" 버튼 → 시퀀스 시작 (자동 진행, 플레이어는 관람)
- "패스트 포워드" 버튼 → 3배속 진행
- 발사 실패 시: "분석 보기" → 실패 원인 + 발사 데이터 상세

#### 비주얼 피드백
- 배경: 실시간 고도에 따른 배경 변화 (지상 → 구름층 → 성층권 → 우주 → 달 궤도)
- Max-Q 구간: 화면 진동 이펙트
- 스테이지 분리: 파티클 이펙트
- 성공 연출: "MISSION SUCCESS" 화면 + 연주 효과음 + 달성 데이터 요약

### 9.5 기술 트리 화면 (Technology Tree Screen)

#### 표시해야 할 데이터
- 6개 브랜치 탭 (구조 / 추진 / 항전 / 열 관리 / 운용 / 달)
- 각 기술 노드: 이름, 아이콘, 현재 상태 (잠금/연구 가능/완료)
- 선택한 기술: 상세 설명, 비용(RP + 재료), 효과, 선행 기술 목록
- 현재 RP 보유량 + 분당 획득률

#### 필요한 인터랙션
- 기술 노드 클릭 → 오른쪽 패널에 상세 정보
- "연구 시작" 버튼 → RP/재료 소비, 연구 진행 바 시작
- 연구 중 다른 기술 클릭 가능 (큐 시스템)

#### 비주얼 피드백
- 잠금 기술: 회색 + 자물쇠 아이콘
- 연구 가능: 파란 테두리 + 빛나는 효과
- 연구 중: 진행 바 애니메이션
- 완료: 금색 체크 + 완료 파티클

### 9.6 마일스톤 & 달 지도 화면 (Milestone & Moon Map Screen)
